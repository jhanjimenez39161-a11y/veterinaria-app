# 🐾 Veterinaria App

Aplicación web desarrollada con React que consume una API pública para mostrar imágenes de perros.  
El proyecto implementa componentes, consumo de API, estilos CSS y diseño responsivo.

---

## 🚀 Funcionalidades

- Consumo de API pública (Dog CEO API)
- Visualización de imágenes de perros
- Botón para cargar más imágenes
- Buscador de imágenes
- Sistema de favoritos ❤️
- Diseño responsive adaptable a móviles y PC
- Interfaz moderna y sencilla

---

## 🛠️ Tecnologías utilizadas

- React
- JavaScript
- HTML5
- CSS3
- Fetch API
- Vite
